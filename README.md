# RTX_DX12_Viewport
 UE Raytracing viewport

A shipping build includes:  

- [ ] ReferencePT.exe
- [ ] nvidia.ico 
- [ ] dxcompiler.dll 
- [ ] dxil.dll 
- [ ] imgui.ini
- [ ] /models/ folder
- [ ] /shaders/ folder

Example command line:
$ ReferencePT.exe -width 1440 -height 1000 -vsync 0 -scene "models\bathroom\LAZIENKA.gltf"
